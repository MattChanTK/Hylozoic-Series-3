CBLA Test Bed

Automatic behaviours generation for interactive art sculptures

Download this project as a .zip file Download this project as a tar.gz file

Overview

The software for the CBLA Test Bed was written to enable both reliable local functions and complex inter-node functions. The main microcontroller used in the current version of the CBLA Test Bed is Teensy. Its code is written in Arduino and compiled using Teensyduino. Each Teensy is connected to computer through USB connection.

System Requirements

  • Windows OS (tested on Windows 8, but should work on XP, Vista and & too)
  • Python 3.3+
  • pyUSB
  • libusb-win32
  • Arduino IDE
  • Teensyduino Add-on
